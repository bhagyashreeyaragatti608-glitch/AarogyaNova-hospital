# AarogyaNova Hospital

Static, GitHub Pages–ready healthcare prediction interface built with HTML, CSS, vanilla JavaScript, Bootstrap and CDN libraries.

For local preview on Windows, double-click `start-server.bat` in File Explorer and keep its terminal window open. It starts the site and opens http://localhost:8080/index.html automatically. Port 8080 avoids conflicts with other projects running on port 8000. Alternatively, publish the repository root with GitHub Pages. The prediction screen uses mock data intentionally; `js/prediction.js` marks the exact location to replace with a production API request.
